Let's Learn About Git Hub!

The Git Hub How-To's:

Below are the steps to create your very own Git Hub:

  • Go to the Git Hub website
  • Create your very own Git Hub account
  • One your account is set to go, pair your index file with your Git Hub
  • To pair your Git Hub with your index files head over to Desktop Github , follow the installation instructions, and your all set!
  • Once your Desktop Github is installed, you will then clone your Git Hub which results in it pairing with your Atom working file, and therefore you have successfully paired your Git Hub!
  • Before proceeding ensure that your Git Hub appears in your Atom working folder
  • Create a new index file and drag it into your Git Hub folder in your Atom working file
  • Save the new index file
  • Sync your new index file to your Git Hub and you will see it appear in green in your Git Hub account
  • The green colour that appears over your text in your Git Hub is simply there to show you the new changes and alterations that have been made. When old content exists, you will see it appear in red above your green text.
  • Congratulations! you have officially set up a functioning Git Hub account!

Want More Info?

  • You should never edit your Git Hub folder or anything within the folder in your Atom working file.
  • Git Hub is really meant to be used by students and classes, like us, or, it is also commonly used in the creation stage of a website, but sites are normally not run through Git Hub.

If you follow the steps listed above, then you will be successful in making your website live!
